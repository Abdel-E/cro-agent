# Cluster-ML

This workspace holds machine-learning and experimentation projects.

## Shopify CRO PoC

The main project is **[`shopify-cro-poc/`](shopify-cro-poc/README.md)** — a proof-of-concept for **conversion-rate optimization (CRO)** on Shopify-style hero content using **multi-armed bandits** (Thompson sampling), optional **multi-stage funnel / journey** APIs, and a **simulator** that logs traffic and produces **charts and heatmaps**.

Start there for setup, how the system works, and where to put **simulation CSVs** vs **exported charts**.
